# 刘白的个人网站

这是一个展示个人作品和技能的响应式个人网站。网站采用现代化的设计风格，具有流畅的动画效果和良好的用户体验。

## 主要特点

- 响应式设计，适配各种设备屏幕
- 现代化UI界面，搭配优雅的渐变色调
- 流畅的滚动和渐入动画效果
- 动态背景Canvas效果
- 项目展示区带二维码悬浮显示
- 回到顶部按钮带进度指示

## 技术栈

### 前端

## 本地运行

1. 克隆仓库
```bash
git clone [仓库地址]
```

2. 打开项目
```bash
cd personal-web-sites
```

3. 运行网站
- 使用本地服务器打开index.html
- 推荐使用Live Server等工具
